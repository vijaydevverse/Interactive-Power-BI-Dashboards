# Power BI Dashboard Portfolio

A collection of interactive dashboards created using Microsoft Power BI Desktop. This repository showcases data visualization, analytics, KPI tracking, and business intelligence projects using real-world datasets across multiple domains including gaming, healthcare, road accidents, and more.

## Dashboards Included

* Gaming Analytics Dashboard
* Healthcare Dashboard
* Road Accident Dashboard
* Sales & Business Insights Dashboard
* Data Visualization Projects

## Tools & Technologies

* Microsoft Power BI Desktop
* Power Query
* DAX
* Data Modeling
* Interactive Visualizations

## Features

* KPI Cards
* Dynamic Charts
* Filters & Slicers
* Drill-through Analysis
* Interactive Reports
* Custom Dashboard Design

## Purpose

This repository is created to showcase Power BI dashboard development skills, data storytelling, and business intelligence techniques using visually appealing and interactive reports.
